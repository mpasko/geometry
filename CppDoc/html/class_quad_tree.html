Opis szczegółowy

Klasa reprezentująca QuadTree, zawiera w sobie metody do przekształcania jego struktury.

Dokumentacja funkcji składowych

void QuadTree::balance_tree ( )

Dzieli drzewo w taki sposób, żeby zrównoważyć wszystkie jego węzły. Metoda wywoływana jest od korzenia i rekursywnie zagłębia się w kolejne dzieci, jeżeli w czasie przechodzenia drzewa nastąpiło podzielenie korzenia, metoda wywoła się ponownie po przejściu całego drzewa, do momentu, gdy nic nie ulegnie zmianie.

void QuadTree::create_extended_neighbours ( )

Otacza węzeł ośmioma sąsiadami tej samej wielkości, poprzez podział węzłów sąsiadujących (o ile nie są już wystarczająco podzielone) z węzłem na którym została wywołana metoda.

void QuadTree::init_mesh ( const list< Point * > *  points_set)

Tworzy początkowy podział drzewa QuadTree poprzez dodawanie do niego kolejnych punktów i odpowiedni podział siatki, tak by nie zanjdowały się w jedynm kwadracie dwa punkty.

void QuadTree::split_too_close_boxes ( )

Dzieli siatkę w taki spoób, żeby nigdy dwa różne punkty nie leżały od siebie bliżej niż 2 * sqrt(2) * a, gdzie a to dlugosc boku kwadratu w ktorym znajduje sie punkt

void QuadTree::surround_with_neighbours_ascending ( )

Dzieli siatkę w taki sposób, że każdy z węzłów zawierający punkt oraz każdy z rodziców tego dziecka, aż do korzenia otoczony jest ośmioma rozszerzonymi sąsiadami tej samej wielkości.
